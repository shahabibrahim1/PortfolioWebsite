# Portfolio Website (React + Vite)

A modern, ultra-minimal portfolio web application built with React and Vite.  
The goal of this project is to present skills, experience, and projects in a clean, fast, and visually polished way, while following good front-end architecture practices.

This repository contains the **source code for the portfolio app**.
